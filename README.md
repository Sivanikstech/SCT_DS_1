# SCT_DS_1

## Task 1: Data Visualization

### Objective

Create a bar chart or histogram to visualize the distribution of a variable such as age or gender.

### Tools Used

* Python
* Pandas
* Matplotlib
* VS Code

### Description

This project demonstrates how to visualize data using:

* Histogram (for continuous data like age)
* Bar chart (for categorical data like gender)

### Output

The graphs show how data is distributed across categories or ranges.
